# 📖 Инструкция по использованию игры "Морской бой" (Async Edition)

## 🚀 Быстрый старт

1. Запустите игру:
   ```bash
   ./run.sh
   ```
   или
   ```bash
   dart run lib/main.dart
   ```

2. Выберите режим игры:
   - `1` - Игра против друга
   - `2` - Игра против ИИ
   - `0` - Выход

3. Введите размер поля (5-10, рекомендуется 10)

4. Корабли размещаются автоматически

5. Вводите координаты в формате: `A1`, `B5`, `J10` и т.д.

## 🎮 Правила игры

- После попадания вы продолжаете ход
- После промаха ход переходит к противнику
- Побеждает тот, кто первым потопит все корабли противника

## 📊 Статистика

Игра показывает:
- Количество попаданий и промахов
- Количество потопленных кораблей
- Точность стрельбы в процентах
- Время игры

## 🎯 Формат координат

Используйте букву (A-J) и число (1-10):
- `A1` - первая строка, первый столбец
- `B5` - вторая строка, пятый столбец
- `J10` - последняя строка, последний столбец

## 🚢 Конфигурация кораблей

Для поля 10x10:
- 1 корабль на 4 клетки
- 2 корабля на 3 клетки
- 3 корабля на 2 клетки
- 4 корабля на 1 клетку

## 🤖 ИИ компьютера

ИИ использует умную стратегию:
- После попадания ищет корабль в соседних клетках
- Автоматически помечает область вокруг потопленных кораблей
- Делает стратегические выстрелы
- Ход ИИ вычисляется в отдельном Isolate (не блокирует UI)

## ⚡ Новые возможности (Async Edition)

### Future
- Корабли размещаются асинхронно с прогресс-баром
- Загрузка статистики предыдущей игры при запуске
- Неблокирующие задержки (Future.delayed вместо sleep)

### Stream
- Все игровые события транслируются в реальном времени
- Уведомления об уничтожении кораблей
- Автоматическое логирование в файл `game_logs/`
- После игры показывается лог последних событий

### Isolate
- Ход ИИ рассчитывается в отдельном изоляте
- Команда "анализ" во время хода — анализ поля в Isolate
- Показывает покрытие, точность, рекомендацию

### Команды во время игры
- Координаты: `A1`, `B5` и т.д. — стрелять
- `анализ` или `analysis` — запустить анализ поля (Isolate)
