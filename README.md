# Flutter приложение с тремя контейнерами

Это Flutter приложение демонстрирует создание трех различных контейнеров с уникальными свойствами и изображением из интернета.

## Особенности

### Контейнер 1 - Градиентный контейнер
- Линейный градиент от синего к фиолетовому
- Скругленные углы (borderRadius: 20)
- Тень для создания эффекта глубины
- Белый текст с жирным шрифтом

### Контейнер 2 - Контейнер с изображением
- Изображение загружается из интернета (picsum.photos)
- Использует CachedNetworkImage для оптимизации
- Полупрозрачный оверлей с текстом
- Обработка ошибок загрузки
- Индикатор загрузки

### Контейнер 3 - Контейнер с иконками
- Оранжевая цветовая схема
- Иконки звезд по бокам
- Центральная колонка с текстом и иконками действий
- Скругленные углы и цветная рамка

## Технические детали

- **Flutter SDK**: ^3.9.0
- **Зависимости**: 
  - `cached_network_image: ^3.3.1` - для кэширования изображений
  - `cupertino_icons: ^1.0.8` - для иконок

## Запуск приложения

1. Убедитесь, что Flutter установлен
2. Перейдите в папку проекта:
   ```bash
   cd container_app
   ```
3. Установите зависимости:
   ```bash
   flutter pub get
   ```
4. Запустите приложение:
   ```bash
   flutter run
   ```

## Свойства контейнеров

### Декорация контейнеров:
- **BoxDecoration** - основной декоратор
- **BorderRadius** - скругление углов
- **BoxShadow** - тени для глубины
- **LinearGradient** - градиентные заливки
- **Border** - цветные рамки

### Расположение элементов:
- **Column** - вертикальное расположение контейнеров
- **Row** - горизонтальное расположение иконок
- **Stack** - наложение элементов (изображение + текст)
- **Positioned** - позиционирование оверлея

### Адаптивность:
- **MainAxisAlignment.spaceEvenly** - равномерное распределение
- **double.infinity** - полная ширина контейнеров
- **Responsive design** - адаптация под разные экраны
